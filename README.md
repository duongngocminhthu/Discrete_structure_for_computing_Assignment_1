# Discrete_structure_for_computing_Assignment_1
# A* Pathfinding Algorithms in C++

Course project for **Discrete Structure (CO1007)**.

## Overview

This project implements the **A* Search Algorithm** in multiple pathfinding scenarios:

* Social Network Pathfinding
* Drone Delivery Route Planning
* Warehouse Robot Navigation
* Evacuation Route Planning

## Concepts Used

* Graph Theory
* A* Search Algorithm
* Heuristic Functions
* Linked Lists
* Path Reconstruction

## Project Structure

* `Algo.cpp` / `Algo.h` – A* algorithm implementations
* `PathNode.cpp` / `PathNode.h` – Linked list data structure for storing paths
* `main.cpp` – Sample test cases
* `AStar_Report.pdf` – Project report

## Technologies

* C++
* STL Vector
* Dynamic Memory Management
